# Variables de Firebase Admin
# Para obtener las credenciales:
# 1. Ve a Firebase Console > Project Settings > Service Accounts
# 2. Click en "Generate new private key"
# 3. Descarga el archivo JSON
# 4. Copia el path completo del archivo aquí

# Agrega esta línea a tu archivo .env existente:
GOOGLE_APPLICATION_CREDENTIALS=./firebase-service-account.json

# O simplemente asegúrate de que el archivo firebase-service-account.json
# esté en la raíz del proyecto backend
