# Backend_EPS

Backend para el proyecto Sistema de Emprendedores (Node.js + Express + Sequelize).

RÃ¡pidos pasos para desarrollo:

- Instalar dependencias:

```bash
npm install
```

- Variables de entorno:

Crear un archivo `.env` con las credenciales de BD y claves (no subirlo al repo):

```
DB_HOST=localhost
DB_USER=root
DB_PASSWORD=...
DB_NAME=eps_db
JWT_SECRET=...
```

- Ejecutar en modo desarrollo:

```bash
npm run dev
```
